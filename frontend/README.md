# Malasiyakart Marketplace

A modern, multi-vendor e-commerce marketplace built for Malaysia.

## Project Architecture

This project is a full-stack MERN (MongoDB, Express, React, Node.js) application.

### Tech Stack

- **Frontend**: Vite, React, TypeScript, Tailwind CSS, shadcn/ui
- **Backend**: Node.js, Express, Mongoose (MongoDB)
- **State Management**: React Context API
- **Authentication**: JWT-based authentication
- **Icons**: Lucide React

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn
- MongoDB Atlas account (or local MongoDB instance)

### Installation

1. **Clone the repository**
   ```sh
   git clone <repository-url>
   cd malasiyakart-marketplace
   ```

2. **Frontend Setup**
   ```sh
   cd frontend
   npm install
   npm run dev
   ```

3. **Backend Setup**
   ```sh
   cd backend
   npm install
   # Create a .env file based on the provided environment variables
   npm start
   ```

## Features

- **Multi-vendor Support**: Sellers can register and list products after admin approval.
- **Admin Dashboard**: Comprehensive stats, user/seller management, and hero banner controls.
- **Modern UI**: Clean, responsive design with glassmorphism and smooth animations.
- **Secure Authentication**: OTP-based login and role-based access control.
- **Product Management**: Category-based browsing, advanced search, and wishlist functionality.
